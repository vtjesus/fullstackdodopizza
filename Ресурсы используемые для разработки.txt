1. SHADCN - сайт с готовыми компонентами различных видов, которые можно редактировать.
А также в нем есть удобные настройки для старта проекта (ShadCn в свою очередь использует radix-ui)

2. Lucide - сайт с кучей иконок, которые можно импортировать прямо в проект с помощью
пакетного менеджера

3. Vercel - сайт для создания баз данных для проекта, от разработчиков next js

4. Prisma - ОРМ между next.js и postgre, которая помогает понять им друг друга и обрабатывать данные
через vercel, после чего возвращать пользователю

5. Table Plus - десктопное приложение для работы с различными базами данных с помощью GUI

6. Resend - сервис, позволяющий тестировать отправку писем на почту с помощью apiKey

БИБЛИОТЕКИ используемые в данном проекте

1. cn - аналог clsx, позволяет комбинировать различные классы, и менять их, используя различные условия.
2. Tailwind - использование css свойств более быстрым способом через специальные классы
3. ReactUse - библиотека с огромным количеством полезных хуков (один из них - useIntersection, для того, 
чтобы в зависимости от расположения на сайте, менять категорию продуктов)
Хуки, которые использовались:
- useIntersection (для определения места на странице при прокрутке, чтобы выбрать нужную категорию)
- useDebounce (для того, чтобы делать отложенный на определенное время в мс запрос)
- useSet (чтобы формировать множество из выбранных чекбоксов, у которых есть несколько функций для работы с множествами, которые хранятся в первой переменной)

4. Zustand - стейт менеджер (Аналог redux)
5. Prisma studio - встроенная в библиотеку prisma утилита, позволяющая работать с бд, используя GUI
6. Postman - библиотека позволяющая делать запросы на сервер и тестировать их
7. qs - библиотека, для удобного формирования query параметров внутри url
8. react-hot-toas - библиотека для вывода красивых всплывающий сообщений сверху
9. react-hook-form - библиотека для работы с формами
10. zod - библиотека, необходимая для создания схем форм, их валидации и дальнейшаяя работа с ними
11. DaData - сервис, позволяющий использовать существующие адреса для тестирования форм с адресами доставками и тд.
12. react-imask - библиотека, позволяющаяя сделать форматирование поля ввода номера телефона (создать маску ввода)
13. NextAuth - библиотека, облегчающая создание процесса авторизации и регистрации пользователя
14. NextJsToploader - библиотека, отображающая вверху страницы прогресс загрузки
